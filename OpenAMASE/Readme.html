Welcome to OpenAMASE

This folder contains executable software and documentation for OpenAMASE

  • Batch files to run AMASE and associated tools are located in the run folder.
  • The LMCP folder contains files pertaining to the LMCP interface and CMASI
  • Documentation regarding the use and design of AMASE is in the doc folder.
  • The source files for this project are contained in the src folder.
  • Dependent libraries are contained in the lib folder.
